Generate a focused drill exercise targeting the #1 weak area.

1. Read `issues/rating.md` and identify the top weak area (highest Total Errors or marked CRITICAL in the Error Tracker table)
2. Read `grammar.md` and `vocabulary.md` for reference material
3. Create a drill file in `exercises/` named `drill-YYYY-MM-DD.md` (using today's date). If one already exists, append a letter suffix (e.g. `drill-2026-02-21b.md`).
4. Generate 10 short, focused exercises on that one weak area. Pick the most appropriate type:
   - **Fill-in-the-blank:** "Il caffè ___ buono." (è / e)
   - **Conjugation completion:** "dormire → loro ___"
   - **Article selection:** "___ studenti vanno a scuola." (i / gli / lo)
   - **Correction:** "Io dormanno bene." → Fix the error.
   - **Choose the correct form:** "Lei (parla / parlano) italiano."
5. Use this exact format:

```
# Drill — YYYY-MM-DD — [Topic Name]

Focus: [description of the weak area being drilled]

1. [exercise prompt]
   >

2. [exercise prompt]
   >

(... 10 exercises total)
```

The user fills in answers on the `> ` lines. Use `/review` to grade the completed drill.
