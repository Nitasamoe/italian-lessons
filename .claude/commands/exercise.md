Generate a new Italian exercise.

**Files to read:**
- `issues/tracking.md` — for weak areas, study recommendations, and progress
- `grammar.md` — read only the index (lines 1–20). Read specific sections only if you need to verify a form or check scope.
- `vocabulary.md` — read only the index (lines 1–18). Read specific sections only if you need to check which words are available.

1. Read the files listed above
2. Create a new exercise file in `exercises/` named with today's date (`YYYY-MM-DD.md`). If one already exists for today, append a letter suffix (e.g. `2026-02-21b.md`).
3. The exercise must contain:
   - **20 sentences in German to translate into Italian**
4. Sentences must:
   - Be at A1 beginner level
   - Be based on the grammar rules in `grammar.md` and vocabulary in `vocabulary.md`
   - Target weak areas from `issues/tracking.md`
   - Mix new material with reinforcement of previous weak spots
   - Adapt difficulty and topics based on the current assessment
   - Include 3–5 multi-clause sentences using subordinating conjunctions (quando, se, mentre, perché, etc.), relative pronouns (che, cui), or infinitive constructions (prima di, senza, per + infinitive) — gradually increase complexity based on what's been demonstrated
5. Use this exact format:

```
# Exercise — YYYY-MM-DD

## German → Italian
Translate the following sentences into Italian.

1. [German sentence]
   >

2. [German sentence]
   >

(... 20 sentences total)
```

The user fills in answers on the `> ` lines.
