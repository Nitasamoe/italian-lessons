## Weak Areas

### Spelling — -mente adverbs (CRITICAL — most frequent error type)
- "rapidament" THREE times across exercises — never gets the final -e
- "rarament" same pattern
- Rule: adjective (feminine form) + -mente = rapidamente, raramente, silenziosamente
- Also general spelling: "carrotte" → "carote", "risposaro" → "riposo", "mane" → "mani"

### Vocabulary confusion (NEW — high impact)
- uva (grapes) vs uova (eggs) — similar looking, totally different
- arancina (rice ball) vs arancia (orange)
- patatine (chips/fries) vs patate (potatoes)
- sempre (always) vs spesso (often) — "oft" = spesso!
- "ante" doesn't exist → "prima di" (before)
- "silentamente" doesn't exist → "piano" / "a bassa voce"

### qualche + singular (persistent — 5th error)
- "qualche volte" AGAIN — this rule is not sticking
- qualche ALWAYS singular: qualche volta, qualche libro, qualche uovo
- For plural use: alcuni/alcune, dei/delle

### quasi mai + non (inconsistent)
- Got it right in #2 ("Non bevo quasi mai") — progress!
- Forgot it in #15 ("Andiamo quasi mai") — still inconsistent
- The "non" is mandatory every time

### Non placement
- "Si sente non bene" → "Non si sente bene"
- "non" ALWAYS goes before the verb (or before the reflexive pronoun)

### Prepositions
- "a colazione" (no article), "al supermercato", "all'aeroporto" (double l before vowel)

## Study Recommendations

1. **TOP PRIORITY: -mente adverbs.** The ending is always -mente, never -ment. rapidaMENTE, raraMENTE. Say it out loud: "ra-pi-da-MEN-TE". Five syllables, not four.

2. **Vocabulary pairs to memorize:**
   - uova = eggs, uva = grapes
   - arancia = orange, arancina = rice ball
   - patate = potatoes, patatine = chips/fries
   - spesso = often (oft), sempre = always (immer)
   - prima di + infinitive = before (bevor)
   - piano / a bassa voce = quietly (leise)

3. **qualche = SINGULAR. Always.** If this still isn't clicking, try: just use "alcune/alcuni" instead. "Alcune uova", "alcuni libri" — then you don't need to remember the singular rule.

4. **non goes BEFORE the verb.** Non si sente bene. Non andiamo quasi mai. Always.

5. Celebrate: reflexives, -ono, è/e, lei/loro, verb stems, collective nouns — all mastered through drills!
